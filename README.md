## 🗺️ What is this?

A production-ready, interactive roadmap for becoming a **complete AI Engineer** — not just a video-watcher, but someone who can build, deploy, and ship real AI systems.

This isn't a random list of links. It's a **curated, ordered curriculum** built from real-world AI engineering requirements, mapping exactly what to learn, in what order, with the best resources for each topic.

## ⚡ The 13 Phases

```
Phase  1 ── Mathematics Foundation       → Statistics, Linear Algebra, Calculus
Phase  2 ── Python & Libraries           → NumPy, Pandas, Matplotlib, Seaborn
Phase  3 ── SQL & Data                   → SQL for Data Science
Phase  4 ── Machine Learning             → Theory + Scikit-learn + Kaggle
Phase  5 ── Deep Learning                → Neural Nets, CNN, RNN, Transformers, PyTorch
Phase  6 ── Large Language Models        → Architecture, Fine-tuning (LoRA/QLoRA)
Phase  7 ── Computer Vision              → OpenCV, YOLO, Segmentation (SAM, U-Net)
Phase  8 ── NLP                          → ML-based NLP, Stanford CS224N
Phase  9 ── MLOps & Deployment           → FastAPI, Docker, CI/CD, DVC, MLflow, AWS/GCP
Phase 10 ── GenAI Applications           → RAG, Agents, LangChain, Graph RAG, Vector DBs
Phase 11 ── Multimodal AI                → Vision + Language, GPT-4o style models
Phase 12 ── Async Python                 → Async/Await for production tool-calling
Phase 13 ── MCP & A2A Protocols          → Model Context Protocol, Agent-to-Agent
```


## 🤝 Contributing

Contributions are welcome and encouraged! Here's how:

1. **Fork** the repo
2. **Create** a branch → `git checkout -b feature/add-resource`
3. **Add** your resource (video or doc) to the correct phase in `App.jsx`
4. **Commit** → `git commit -m "feat: add XYZ resource to Phase N"`
5. **Push** → `git push origin feature/add-resource`
6. **Open a Pull Request** — describe what you added and why it belongs

### What makes a good contribution?
- ✅ Free resource (YouTube video, official docs, open course)
- ✅ Placed in the correct phase with correct topic
- ✅ High quality — not just any video, but the *best* one for that topic
- ✅ Adds something genuinely missing, not a duplicate
